# CLI-Gantt-Chart-Generator (C)

---

A command-line Gantt chart generator written in C.

## Features

- Create, edit, and visualise project tasks  
- Includes a preloaded test project for quick demonstration  
- Detects circular dependencies using recursion  
- Calculates the critical path of a project  
- Displays tasks in a clear, tabular Gantt chart format  
- Built using structs, enums, recursion, and a modular design


---

## Installation

---

git clone https://github.com/kingmbachu/CLI-Gantt-Chart-Generator.git
